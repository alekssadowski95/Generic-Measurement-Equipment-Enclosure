# Open-Source Measurement Equipment

Generic measurement equipment framework (enclosure, PSU, display, ...) to be used with:
- Digital Oscilloscopes
- Waveform Generators
- Spectrum Analyzers
- RF Generators
- Vector Network Analyzers
- RF Switch Systems
- Digital Multimeters
- DC Electronic Loads
- Power Supplies
